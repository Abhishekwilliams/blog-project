About Me

Thanks for visiting my blog! Check out all this cool stuff about me!

Hello! I'm Abhishek Williams, a passionate Full Stack Developer with expertise in Python, C++, and various web technologies. I have a strong background in AI and Machine Learning, having completed my Ontario Graduate Certificate from Lambton College, Toronto. My technical skills include Angular, Vue.js, JavaScript, TypeScript, Django, and REST APIs.

I have professional experience as an Angular/Vue.js Programmer/Developer at Unified Infotech, where I worked on creating asset management software and developing various user interface components. I also interned at NIC, MeitY, Govt. of India, where I contributed to a web application to track the grain distribution system throughout India.

  • Orpheus Sabha: A website for a college public speaking platform. It aims to create awareness and outreach programs about public speaking platforms.
    Skills: AngularJS, Django, HTML, CSS, TypeScript, SQL, MySQL.
  • Connect 4 Game: A simple game developed using JavaScript, CSS, and HTML. Players take turns dropping colored discs into a grid to connect four in a row.
    Skills: HTML, CSS, JavaScript, Git.

    Skills: Django, JavaScript, HTML, CSS, Git.
